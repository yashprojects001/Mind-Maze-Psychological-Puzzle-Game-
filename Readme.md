# 🧠✨ Mind Maze — Psychological Puzzle Game

> *"The maze isn’t broken… your mind is."* 👁

---

## 🚀 Overview

**Mind Maze** is a **frontend-only psychological puzzle game** built using:

🔹 HTML
🔹 CSS
🔹 JavaScript

Navigate through a constantly shifting maze that **challenges perception, memory, and focus**.

---

## 🎮 Features

🧩 **Dynamic Maze Generation**
Every game creates a new unpredictable maze.

👀 **Psychological Twist**
The maze may suddenly change while you play…

⏱ **Real-Time Timer**
Track how long you survive.

⭐ **Score System**
Reach the goal → increase your score.

🌌 **Dark Glitch UI**
Immersive neon + cyber aesthetic.

🎯 **Goal-Based Gameplay**
Reach the red square to win each round.

## 🧠 Game Logic

🟢 Player starts at top-left
🔴 Goal is at bottom-right
⬛ Walls block movement
🧠 Maze may randomly regenerate (psychological effect)
